## 🦀 Maryland Flash Card Quiz

A beginner-friendly command-line quiz about the state of Maryland!

### Files
| File | Description |
|---|---|
| `flashquiz-maryland.py` | The quiz program |
| `questions.json` | The flash card questions and answers |

### How to Run

1. Make sure you have Python 3 installed:
   ```bash
   python3 --version
   ```

2. Run the quiz from the terminal:
   ```bash
   python3 flashquiz-maryland.py
   ```

3. Type your answer and press **Enter** for each question.

4. Your final score is shown at the end. Good luck! 🏁

### Requirements
- Python 3 (no extra packages needed)
- `questions.json` must be in the same folder as `flashquiz-maryland.py`
